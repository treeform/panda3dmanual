Enhanced mouse navigation is only available when 'Direct Tools' are enabled.  Information on enabling 'Direct Tools' is available in the [[Panda Tools]] section.

Direct Tools gives more functionality to the middle mouse button. First, clicking the middle mouse button changes the pivot point it uses to rotate around the environment. The middle mouse button will also move the camera depending on where the cursor is on the screen.

<center>[[Image:directtools2.jpg]]</center>

<center><table>
<tr><td>Middle Mouse Click</td><td>Sets pivot point for rotating around the world</td></tr>
<tr><td>Middle Mouse + Middle Region</td><td>Move camera parallel to ground</td></tr>
<tr><td>Shift + Middle Mouse + Middle Region</td><td>Move camera vertically</td></tr>
<tr><td>Middle Mouse + Edge Region</td><td>Rotate camera around pivot point</td></tr>
<tr><td>Middle Mouse + Corner Region</td><td>Roll camera around pivot point</td></tr>
<tr><td>Shift + Middle Mouse + Edge Region</td><td>Changes pitch of the camera</td></tr>
</table></center>

The left mouse button is now used to select and manipulate objects in the environment. Once an object is selected, it may be moved and rotated.

<center><table>
<tr><td>Left Mouse Click</td><td>Select an object</td></tr>
<tr><td>Left Mouse + Middle Region</td><td>Move object vertically</td></tr>
<tr><td>Shift + Left Mouse + Middle Region</td><td>Move object parallel to ground</td></tr>
<tr><td>Left Mouse + Edge Region</td><td>Rotate object around its pivot point</td></tr>
<tr><td>Left Mouse + Corner Region</td><td>Roll object around its pivot point</td></tr>
<tr><td>Control + Left Mouse</td><td>Rescale the model</td></tr>
</table></center>
